# Deliverables Checklist (提交物清单)

依据《竞赛题2_运动识别.md》中的描述，建议提交时包含以下内容：

## 1. 可执行文件（用于组委会隐藏测试集评测）

- `submit_runner/`（PyInstaller onedir 产物，或 Windows 下的 `.exe` 及其依赖）
- 运行后能在当前目录生成 `submission.xlsx`
- 程序中包含并引用组委会提供的 `DataReader` 与 `DataOutput` 示例接口代码

## 2. Python 源码（用于复核与复现）

- 训练代码（如何从训练集数据复现模型）
- 推理代码（如何从 `test_data/*.txt` 生成 `submission.xlsx`）
- readme（一步一步说明复现流程与环境依赖）

## 3. 总结报告（用于答辩与展示）

组委会给出的模板字段：
- 背景与基础、场景价值、项目亮点
- 整体目标、技术创新点
- 技术可行性分析、技术细节
- 自验证分析（user-holdout 结果、误判案例、迭代措施等）
- 项目总结与展望、参考文献

## 4. 提交提醒

- 官网通常只能提交一次且提交后不可更改，请确保最终版本完整可运行。
